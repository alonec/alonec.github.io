面向对象的三大基本特性

发表于 2018-06-10 | 分类于 java | |
字数统计: | 阅读时长 ≈
封装，继承及多态。还有重载和重写。

一、概述

世间一切皆为对象

现实世界中任何实体都可归属于某类事物，任何对象都是某一类事物的实例。

二、java面向对象的三大基本特性

1、封装

封装指的是讲对象内部状态隐藏在对象内部，不允许外部程序直接访对象内部信息，只能通过类提供的方法实现对内部信息的操作访问。

2、继承

继承是指一个新的类继承原有类的基本特征，并增加了新的特性。

3、多态

多态是指允许不同类的对象对同一消息作出响应，即同一消息可以根据发送对象的不同而采用多种不同的行为方式。

三、重载和重写

1、重载

  • 重载发生在同一个类内的两个或多个方法。

重载必须满足的条件：

  1. 方法名必须相同；
  2. 方法返回类型，修饰符可不同；
  3. 参数类型或个数或顺序要不同。

2、重写

  • 重写又叫覆盖，发生在父类和子类之间。

重写必须满足的条件：

  1. 方法名相同；
  2. 返回类型相同，修饰符可以不同，但是修饰符：子类>=父类；
  3. 参数列表要一样。
